# 開発用README

## 概要

**ウェザーバスター 〜CLEAR THE SKY〜** は、クセのある天候現象モンスターを倒して、荒れた空を晴れに戻す3Dアリーナシューターです。

プレイヤーは「ウェザーブレイカー」として、豪雨・落雷・竜巻・梅雨・台風・吹雪などの異常気象と戦います。  
敵を倒すと空が晴れ、勝利演出として `CLEAR SKY!` が表示されます。

## 世界観

異常気象モンスターによって、各地の空は荒れ続けています。

プレイヤーは、天候現象に干渉できる特殊装備を持つ **ウェザーブレイカー** として観測区域に出撃し、天候モンスターのコアを破壊して空を晴れに戻します。

### 裏設定

主人公は梅雨に強い恨みを持っています。

理由は明確に語りすぎず、ゲーム内の小ネタや図鑑、戦闘前テキストで少しずつ出していく想定です。

例:

> 梅雨だけは、許さない。

## 勝利条件

敵天候のHPを0にすること。

```txt
敵撃破
  ↓
雲が割れる
  ↓
空が青くなる
  ↓
CLEAR SKY!
```

## 敗北条件

- プレイヤーHPが0になる
- 制限時間を超える
- 特殊ミッションの失敗条件を満たす

敗北時表示:

```txt
WEATHER OVER
```

## 敵天候

### 曇り

チュートリアル・練習用の敵。

特徴:

- 移動が遅い
- 攻撃が弱い
- 視界を少しだけ曇らせる
- 倒すと空が晴れる感覚を学べる

役割:

ゲームの基本操作と勝利演出を覚えるための存在。

### 豪雨

水と視界妨害を使う基本敵。

特徴:

- 雨弾を連射する
- 画面に水滴エフェクトを出す
- 地面に水たまりを作る
- 水たまりに入ると移動速度が下がる

役割:

移動・回避・射撃を組み合わせて戦う最初の本格敵。

### 落雷

落雷と予兆攻撃を使う敵。

特徴:

- 地面に落雷予告マーカーを表示
- 数秒後に落雷
- 当たると大ダメージ
- 攻撃前に音と光で予兆を出す

役割:

回避と位置取りを学ぶ敵。見た目も派手で映像映えしやすい。

### 竜巻

吸引・吹き飛ばしを使う敵。

特徴:

- プレイヤーを吸い寄せる
- 吹き飛ばし攻撃をする
- 弾道や移動に影響を与える
- フィールド内を移動し続ける

役割:

位置取り重視の敵。将来的な追加候補。

### 梅雨

状態異常・長期戦型の敵。

特徴:

- 湿気フィールドを展開
- 移動速度低下
- 視界悪化
- 時間経過で不利になる
- 梅雨キラーが有効

役割:

主人公の因縁敵。ゲームの個性を強く出す重要キャラ。

### 吹雪

視界・移動妨害型の敵。

特徴:

- 視界を白くする
- 足場を滑りやすくする
- 凍結状態を付与する
- 移動やエイムを妨害する

役割:

雪・冬系ステージ用の敵。将来的な追加候補。

### 台風

ボス候補。

特徴:

- 豪雨・落雷・竜巻の複合型
- 吸引
- 瓦礫飛ばし
- 竜巻生成
- HP減少で攻撃パターン変化
- 撃破時の晴天化演出を最も派手にする

役割:

大型ボス。最終的に「ウェザーバスターらしさ」を最も表現する敵。

## スキル設計

```txt
キャラ固有スキル: 常時発動
武器スキル: 命中でゲージが溜まり、任意発動
```

### キャラ固有スキル

キャラクターごとに常時発動するパッシブスキル。

| スキル名 | 効果 |
| --- | --- |
| 晴天体質 | 天候デバフを少し軽減 |
| 低気圧耐性 | 鈍足・視界妨害を軽減 |
| 風読み | 竜巻・吹雪の移動妨害を軽減 |
| 避雷感覚 | 落雷予告が少し早く見える |
| 梅雨嫌い | 梅雨系の敵に与えるダメージ増加 |

### 武器スキル

武器ごとに設定されたアクティブスキル。敵に弾を命中させることでゲージが溜まり、ゲージ最大時に `Q` キーで発動できます。

| 武器 | スキル名 | 効果 |
| --- | --- | --- |
| ウェザーガン | 前線押し戻し | 敵をノックバック |
| クリアスカイガン | 晴天バースト | 周囲の天候効果を一時的に弱める |
| 梅雨キラー | 除湿フィールド | 湿気・鈍足効果を解除 |
| 高気圧ライフル | 高気圧集中 | 一定時間、精度と威力上昇 |
| 雷雲ガンビット | 自動観測射撃 | ドローンが自動射撃 |

## アイテム

アイテムはフィールド上に落ちており、近づくと拾えます。保持したアイテムは `E` キーで使用します。

基本ルール:

```txt
フィールドに落ちている
近づくと拾える
最大保持数を設定
Eキーで使用
```

MVPでは最大1個保持でも成立します。

### アイテム候補

| アイテム名 | 効果 | 対応天候 |
| --- | --- | --- |
| 晴れ薬 | HP回復 | 全般 |
| デコイ傘 | 敵の攻撃を引きつける | 豪雨・落雷 |
| 避雷針 | 次の雷攻撃を1回無効化 | 落雷 |
| 除湿機 | 鈍足・湿気を解除 | 梅雨 |
| 気圧安定剤 | スキルゲージ上昇 or クールタイム短縮 | 全般 |
| 晴天ビーコン | 周囲の天候効果を一時解除 | 全般 |
| レインブーツ | 水たまりの鈍足を無効化 | 豪雨 |
| 風よけマント | 吹き飛ばし軽減 | 竜巻・台風 |
| スノーゴーグル | 吹雪の視界不良を軽減 | 吹雪 |

## ステージ

### 実験場

MVP用ステージ。

特徴:

- 無機質
- ラボっぽい
- シンプル
- 障害物を置きやすい
- 天候現象を出しても違和感が少ない

### 荒野

竜巻・台風と相性がよいステージ。

特徴:

- 地形に起伏がある
- 視界が広い
- 風・砂埃・雷などの演出が映える

### 街

豪雨・落雷と相性がよいステージ。

特徴:

- 建物を遮蔽物にできる
- 路面の水たまり演出がしやすい
- 雷や雨の演出が映える

### 貨物線

荒野系ステージの派生。

特徴:

- コンテナや列車を遮蔽物にできる
- 直線的な戦闘が作りやすい
- 竜巻・豪雨と相性がよい

### オフィス

梅雨と相性がよいステージ。

特徴:

- 屋内
- 湿気
- 壊れた空調
- 曇った窓
- 除湿機や傘立てなどの小物を置ける

## 現在地の気象情報連動

将来的に位置情報や天気APIを使って現在地の気象情報を取得します。

ただし、ステージ天候を完全に現在地連動にすると敵天候演出と競合しやすいため、まずは支給アイテム・バフ・今日のミッションを変える用途を優先します。

### 優先案

```txt
現在地の天気に応じて、支給アイテム・バフ・今日のミッションを変える
```

### 例

| 現在地の天気 | 効果 |
| --- | --- |
| 晴れ | 晴天バーストのゲージ上昇量アップ |
| 曇り | 視界妨害を少し軽減 |
| 雨 | レインブーツを初期支給 |
| 雷 | 避雷針を初期支給 |
| 雪 | スノーゴーグルを初期支給 |
| 強風 | ダッシュ消費軽減 |
| 高温 | 晴れ薬の回復量アップ |

表示例:

```txt
現在地の観測データ: 雨
支給アイテム: レインブーツ
今日の気象補正: 水たまり耐性 +20%
```
