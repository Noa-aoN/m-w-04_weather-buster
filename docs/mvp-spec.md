# MVP仕様

## 目的

MVPの目的は、次の体験を成立させることです。

```txt
敵天候を倒す
  ↓
空が晴れる
  ↓
CLEAR SKY!
```

この段階では、シューターとしての網羅性よりも「晴天化の達成感」を優先します。

## 対象範囲

### ステージ

- 実験場 1ステージ

### 敵

- 曇り
- 豪雨
- 落雷

### 武器

- ウェザーガン
- クリアスカイガン
- 梅雨キラー

### スキル

- キャラ固有スキル
- 武器スキル

### アイテム

- 晴れ薬
- デコイ傘
- 避雷針
- 気圧安定剤

### 結果画面

- 撃破時間
- 命中率
- 被ダメージ率
- 晴天化スコア
- ランク

## プレイヤー要件

### 操作

- `WASD`: 移動
- `Mouse`: 視点移動
- `Left Click`: 射撃
- `Space`: ジャンプ
- `Shift`: ダッシュ
- `R`: リロード
- `Q`: 武器スキル発動
- `E`: アイテム使用
- `Esc`: ポーズ

### 必須機能

- 3D空間を自由に移動できる
- 視点操作で敵を狙える
- 射撃で敵にダメージを与えられる
- 被弾で耐候値が減る
- アイテムを拾って使える
- 武器スキルのゲージ蓄積と発動ができる

## 戦闘要件

### 共通

- 敵は耐久値を持つ
- 敵撃破で勝利状態へ遷移する
- プレイヤー耐候値が0で敗北
- 制限時間超過で敗北

### 敵ごとの最低要件

#### 曇り

- 緩やかに接近する
- 低威力攻撃を行う
- 軽い視界妨害を発生させる

#### 豪雨

- 雨弾を連射する
- 水滴系の視覚妨害を発生させる
- 水たまりを生成し、踏むと移動速度低下

#### 落雷

- 落雷予兆マーカーを地面に出す
- 一定時間後に落雷する
- 高威力攻撃として機能する

## スキル要件

### キャラ固有スキル

- 常時発動パッシブ
- バトル中の入力を必要としない
- 主に耐性・視認性・特定天候への特効で差別化する

### 武器スキル

- 命中で気圧ゲージが増加する
- ゲージ最大時に `Q` で発動できる
- 発動後にゲージを消費する
- 武器ごとに明確な役割差を持つ

## UI要件

- 左上: プレイヤー耐候値
- 右上: 敵耐久値 / 敵天候名
- 中央: 照準
- 下部: 武器名 / 残弾 / アイテム / 気圧ゲージ
- 中央上: 制限時間
- 勝利時: `CLEAR SKY!`
- 敗北時: `WEATHER OVER`

## スコア要件

晴天化スコアは以下を軸に算出します。

- 撃破時間
- 命中率
- 被ダメージ率
- 残り耐候値
- スキル使用回数
- アイテム使用数

## 非対象

- ログイン
- オンラインランキング
- キャラ育成
- マルチプレイ
- 複雑な装備ビルド
- 高度な敵AI
- セーブデータ
- 課金要素

## 完了条件

MVP完了は、次を満たした時点とします。

- タイトルから戦闘開始まで遷移できる
- 曇り・豪雨・落雷のいずれかを選んで戦える
- プレイヤーが移動・射撃・回避できる
- 敵の攻撃ギミックが成立している
- 撃破で空が晴れ、`CLEAR SKY!` を表示できる
- 敗北条件と結果画面が動作する
