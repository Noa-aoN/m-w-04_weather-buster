# 仕様 / 設計判断

## アプリ概要

**ウェザー・バスターズ 〜CLEAR THE SKY〜** は、異常気象を擬人化した「天候性侵害体(略して 天侵体)」を撃破して空を晴れに戻す 3D アリーナアクションです。撃破時に空が割れて青空が戻る `CLEAR SKY!` 演出を体験のゴールに置いています。

## ゲームフロー

1. ホーム → 出撃ブリーフィングでバスター / ウェポン / ステージ / 天侵体 / 難易度を選択
2. ゲーム開始 → カウントダウン後に戦闘開始
3. 戦闘中はコア(発光部)を狙うとダメージ大、ボディヒットは大幅減衰
4. 気圧ゲージ満タンで `Q` の **ウェポンスキル** 発動
5. 撃破で `CLEAR SKY!` 演出 → 結果画面でランク・スコア確認

## 戦闘の主要要素

- **コア / ボディ判定**: コア命中で約 5.5 倍の与ダメ差。エイム精度がダメージレートを決める
- **硬直フェーズ**: ボス HP 75% / 40% を切ると 3 秒間動きが止まり、攻撃チャンスタイム
- **子分召喚**(豪雨 / 梅雨): HP 80% / 50% / 25% で「曇り」型の小型天侵体を 1〜3 体召喚。難易度 Lv1-2 ではスキップ。子分が居る間ボスは離れて攻撃頻度激減・微回復
- **撃破演出**: ボス本体が粉砕シャードに分解 + カメラが空を見上げる(3.6 秒のパン)。BGM は victory に切替

## ウェポン

ウェザーガン / クリアスカイガン / 梅雨キラー(対湿気特効) / 雷除けライフル(対落雷特効) / 凍結カノン(対吹雪・竜巻特効) / 風切ブレード(近接、対竜巻・台風特効)

各ウェポンに **武器スキル**(気圧ゲージ消費の高威力バースト)が紐づく。

## 操作(戦闘中)

- 移動:`W A S D`、視点:マウス
- 攻撃:左クリック、リロード:右クリック / `R`
- ウェポンスキル:`Q`、気象シールド:`B`(長押し)
- アイテム:`1`〜`4`、ジャンプ:`Space`、ダッシュ:`Shift`、ポーズ:`Esc`

## 主な設計判断

- 戦闘の純粋関数は `src/game/combatRules.ts` に集約。Vitest で単体テスト
- 状態は Zustand(`src/game/battleStore.ts`)。フィールド追加時は `BattleState` 型・`baseLoadout()` 初期化・`set()` の 3 箇所を必ず揃える
- 3D は React Three Fiber、HUD は React + CSS、戦闘空間内 HP バーは drei `<Html>` で 3D 座標追従
- BGM は Web Audio API のシーンベース合成(title / battle / victory / defeat)。SFX は CC0 サンプル + シンセ合成のハイブリッド
