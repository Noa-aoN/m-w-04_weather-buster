# LP / 紹介文用README

## タイトル

**ウェザーバスター 〜CLEAR THE SKY〜**

## 一言紹介

荒れた空を撃ち抜き、晴れを取り戻す3Dアリーナシューター。

## 紹介文

**ウェザーバスター 〜CLEAR THE SKY〜** は、豪雨・多雷・竜巻・梅雨など、プレイヤーの行動を妨げる異常気象モンスターを撃破し、空を晴れに戻していくゲームです。

プレイヤーは特殊装備を持つ **ウェザーブレイカー** として観測区域へ出撃します。敵ごとに異なるギミックを見極め、攻撃を避け、撃ち抜き、最後は空そのものを取り戻します。

最大の見どころは、敵を倒した瞬間に雲が割れ、青空とともに `CLEAR SKY!` が表示される勝利演出です。

## コンセプトコピー案

- 荒れた空を撃ち抜き、晴れを取り戻せ。
- 豪雨も、雷も、梅雨も、撃ち抜け。
- 空を奪う異常気象を、観測区域から排除せよ。
- 倒すたびに、世界が晴れていく。

## 体験の特徴

- 天候現象そのものを敵として戦う、見た目にも分かりやすいテーマ
- 視界妨害・鈍足・落雷予兆など、敵ごとに個性のある攻撃
- キャラ固有パッシブと武器スキルの組み合わせによるビルド性
- 撃破時に空が晴れる、明快で気持ちのよい達成演出

## スキル設計の魅力

本作のスキルは、次の2系統で構成します。

```txt
キャラ固有スキル: 常時発動パッシブ
武器スキル: 命中でゲージが溜まり、任意発動
```

この分け方により、キャラ選択・武器選択・戦闘中の判断がそれぞれ別の意味を持ちます。
