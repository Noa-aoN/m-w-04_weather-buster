PBR Field Textures
==================

These ground / floor PBR texture sets come from AmbientCG (formerly
"CC0 Textures"). Each set originally shipped at 2K JPG with Color /
NormalGL / Roughness / AmbientOcclusion / Displacement / NormalDX maps.

For this project we use only Color / NormalGL / Roughness / AO and have
downsampled to 1024 px JPG quality 80 to keep web load times reasonable.
