# ウェザー・バスターズ 〜CLEAR THE SKY〜

荒れた天候を撃ち抜き、空を晴らす 3D アリーナアクションです。  
プレイヤーは「ウェザー・バスター」として、豪雨・落雷・竜巻・梅雨・吹雪などの **天候性侵害体（略して 天侵体）** を制圧します。  
撃破した瞬間、その地点だけ雲が割れて青空が戻る ―― それが本作のゴール『CLEAR SKY』です。

URL: https://noa-aon.github.io/m-w-04_weather-buster/

## 動機

「悪天候を倒してすっきりする・してもらう・立ち向かうために」
- 悪天候が苦手なので、倒してみたかった
- 3Dアセット活用・ゲーム作成のVive Codingを試してみたかった

## 見どころ

とりあえず追加したもの・ところどころ開発中にはなります
- モンスターごとに異なる3Dアセット・攻撃パターン・ステージ演出
- ウェポンごとに異なる3Dアセット・パラメータ（**武器スキル** や **弱点特効**）
- 複数キャラクター（まだ2人）ごとに異なる3Dアセット・パラメータ・フレーバーテキスト
- ボス戦中の **子分召喚**(豪雨 / 梅雨)演出
- 撃破時の空を見上げるカメラパン
- 図鑑

## 遊び方

1. ホーム画面の 左のメニューや **出撃ブリーフィング** でバスター・ウェポン・ステージ・敵・難易度を選択
2. **ゲーム開始** で出撃 → カウントダウン → 戦闘開始
3. 敵の **コア**(発光部)を狙うとダメージ x3、ボディヒットは大幅減衰
4. 気圧ゲージが満タンで `Q` 武器スキル発動
5. HP を削り切れば **CLEAR SKY**

## 動作環境

WebGL2 が動く最近のブラウザ(Chrome / Edge / Safari / Firefox)で動きます。3D アリーナを毎フレーム描画するため、ある程度のスペックがあった方が快適です。

| | 推奨 | 最低 |
| --- | --- | --- |
| PC | デスクトップ GPU 搭載機 / 2020 年以降の MacBook Pro クラス | 統合 GPU の薄型ノート(Intel Iris Xe・Apple M1 以降) |
| スマホ | iPhone 13 以降 / Snapdragon 8 系の Android | iPhone 11 / ミドルレンジ Android |
| ブラウザ | Chrome / Edge 最新版 | WebGL2 が動く最新ブラウザ |
| 解像度 | フル HD 程度 | - |

カクつきが気になる場合は、ブラウザ拡張(特に映像系)を切る・他のタブを閉じる・OS の省電力モードを切る、で改善することがあります。

## 登場要素

### 天侵体(敵)

- **曇り** … チュートリアル級。動きが緩く、入門用
- **豪雨** … 視界妨害 + 連続攻撃。子分召喚あり
- **落雷** … 高威力の落雷。直撃は致命
- **竜巻** … 吸引と接近圧
- **梅雨** … 鈍足を含む状態異常。子分召喚あり
- **吹雪** … 視界と動きを乱す氷雪系
- **台風** … 複合型(調整中)

### ウェポン

ウェザーガン / クリアスカイガン / 梅雨キラー / 雷除けライフル / 凍結カノン / 風切ブレード(近接)※開発中

### バスター(プレイアブル)

- **ノア(NOA)** ― 機動バランス型。与ダメ +10% / 移動 +10%
- **サカ(SAKA)** ― 重装アタッカー型。与ダメ +30% / 被ダメ -15% / 気圧 +30% / 移動 -20%

## 技術構成

| レイヤ | 採用 |
| --- | --- |
| ビルド / 開発 | Vite |
| UI | React 19 + TypeScript |
| 3D 描画 | Three.js + @react-three/fiber + @react-three/drei |
| 状態管理 | Zustand |
| テスト | Vitest |
| デプロイ | GitHub Pages(GitHub Actions) |

実装の特徴的な点:

- 戦闘ロジック(ダメージ計算、装填、コンボ、ステイガー閾値、子分スポーン)はピュア関数として `combatRules.ts` に分離 → Vitest で純粋テスト
- `OrbitControls` を活用した図鑑類のドラッグ回転 / ホイールズーム可能なプレビュー
- BGM はシーン別の Web Audio API 合成(`audio.ts`)。SFX は CC0 サンプル + シンセ合成のハイブリッド
- HUD は React + CSS、戦闘空間内の HP バーは drei の `<Html>` で 3D 座標に追従
